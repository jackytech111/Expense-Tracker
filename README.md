# Money Tracker

A personal finance management web app built with Vanilla JavaScript.

## Features
- Add & delete transactions
- Income & expense calculation
- Persistent data using LocalStorage
- Dynamic UI rendering

## Tech Stack

- CSS
- JavaScript (ES6)
